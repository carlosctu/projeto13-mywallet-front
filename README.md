## About The Project

<p align="justify"> MyWallet is kind of a Bank App, made it in React and Node.JS, to track your daily incomes and outcomes.</p>

 <img width="300px" height="480px" src="https://user-images.githubusercontent.com/72350887/189780243-7575a2b8-600b-49df-9230-25ff82cad5ff.gif" alt="TrackIt GIF"/>
</div>


## Main functionalities

- Register/Sign up
- Login/Sign in
- Show your transactions and balance
- Add new incomes or outcomes

## How to run

1. Clone this repository
2. Install dependencies
```bash
npm i
```
3. Run the UI with
```bash
npm run start
```
4. Finally access http://localhost:5000 on your favorite browser (unless it is Internet Explorer. In this case, review your life decisions)
